A lot of new CLI tools depend on a `golang` install.

Installing on Ubuntu

Assuming that you're running as the vagrant user {% highlight bash %} mkdir /home/vagrant/gocode echo "export PATH=/home/vagrant/gocode/bin:$PATH >> /home/vagrant/.bashrc" source /home/vagrant/.bashrc sudo apt-get install golang {% endhighlight %}

Ensuring it works

{% highlight bash %} go --help {% endhighlight %}
